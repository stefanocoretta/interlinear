To create an interlinear gloss, use a fenced Div block (\texttt{:::})
with class ``ex''.

\begin{itemize}
\tightlist
\item
  Each line of the interlinear gloss should start with ``\textbar{}''.
\item
  Interlinear lines that start with ``\textbar{} -'' are treated as
  preambles/original text (i.e.~they are not aligned).
\item
  The last interlinear line is treated as the free translation (single
  quotes are added automatically).
\end{itemize}

\phantomsection\label{ex-kl}
\ex
\begingl
\glpreamble  DaH mojaqmeyvam DIvuSnISbe’ ’e’ vIHar //
\gla DaH mojaq-mey-vam DI-vuS-nIS-be’ ’e’ vI-Har //
\glb now suffix-PL-DEM 1PL.A.3PL.P-limit-need-NEG that 1SG.A.3SG.P-believe //
\glft `I believe that we do not need to limit these suffixes now.' //
\endgl
\xe

To create an example without free translation (and without the original
text line):

\phantomsection\label{ex-sp}
\ex
\begingl
\gla el perrito está comiendo //
\glb the {little dog} is eating //
\endgl
\xe

Examples can also be simple paragraphs, with no interlinear glossing.

\ex
Trying this.
\xe
